# Gallery Engine Snapshots

Drop real engine screenshots into this folder using these filenames:

1. `engine-research-dashboard.png`
2. `engine-governance-replay.png`
3. `engine-tour-panel.png`
4. `engine-report-view.png`

Optional alternatives: `.jpg`, `.jpeg`, `.webp` (if you use a different extension, update `/demo/gallery/index.html`).

Recommended export size:
- 1920x1080 or 1600x900
- Keep UI text readable
- Avoid any private identifiers before exporting

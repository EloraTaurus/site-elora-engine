## Elora Fabric v1 — Codex Build Guide

This bundle defines the minimum architecture required for Elora Engine
to evolve into Elora Fabric.

Elora Fabric is a policy-governed control plane for executing AI-initiated
actions across isolated, untrusted environments.

## What Codex Must Do
- Implement exactly what is specified
- Follow task order
- Prefer safety over convenience
- Do not invent agent autonomy
- Do not bypass justification or guardrails

## What Success Looks Like
A WordPress draft is created via:
- action proposal
- mandatory justification
- guardrail enforcement
- isolated worker execution
- complete audit trail

If this works, Elora Fabric v1 is complete.

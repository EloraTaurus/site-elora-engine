# Public Documentation Pack

This folder contains public-safe copies of selected documentation from `engine/`, `fabric/`, and `workerhost/`.

The pack is curated to explain Elora as a research engine with governance capability. It includes public-safe references to:

- proposal-to-commit governance controls
- deterministic replay and admissibility evidence
- ERIS runtime intelligence
- non-neural behaviour-detection research direction
- bounded learning, continuity, and operator review surfaces

It intentionally omits implementation depth that would make private operational paths directly reproducible.
